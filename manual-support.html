프로그램 소개

개요

시스템동바리 구조계산 프로그램은 건설 현장에서 사용되는 시스템 동바리의 구조 안전성을 자동으로 계산하는 웹 기반 프로그램입니다.

주요 특징

  • 실시간 자동 계산 입력값 변경 시 즉시 구조계산 수행
  • 다중 구조물 지원 슬래브, 일반 보, 데크 보 동시 계산
  • 3D 시각화 구조물의 3차원 모델링 제공
  • 보고서 자동 생성 참고용 구조계산서 즉시 출력

지원 구조물

  1. 슬래브 - 100~3500mm 두께
  2. 일반 보 - 300×300 ~ 1200×2000mm
  3. 데크 보 - 데크슬래브 포함 보 구조
참고사항
본 프로그램의 모든 계산 결과 및 권장 수치는 참고용입니다.
실제 적용 시에는 현장 조건을 고려하여 적절히 조정하시기 바랍니다.
중요 안내사항
본 프로그램은 참고용이며, 현장 적용 시 반드시 구조기술사의 검토가 필요합니다.

시작하기

프로그램 실행

  1. 웹 브라우저에서 프로그램 파일 실행
  2. Chrome, Edge, Firefox 등 최신 브라우저 권장

초기 화면

구조 타입 선택
  • 🏢 슬래브 - 바닥 슬래브 구조
  • 🏗️ 일반 보 - 표준 보 구조
  • 🏭 데크 보 - 데크플레이트가 있는 보 구조

인터페이스 구성

  • 왼쪽 모니터 - 입력 파라미터 및 계산 결과
  • 오른쪽 모니터 - 시스템 다이어그램 및 상세 정보
  • 상단 툴바 - 저장, 불러오기, 인쇄 등 기능

주요 기능

자동 계산 시스템

프로그램은 입력값에 따라 다음 항목을 자동으로 계산합니다:

참고용 안내
아래 표시된 간격의 최소값과 최대값은 참고용입니다.
실제 현장 조건에 따라 상이할 수 있으므로, 반드시 현장 여건을 고려하여 적용하시기 바랍니다.
항목 설명 참고 범위*
장선 간격 거푸집 하부 지지재 130~400mm
멍에 간격 장선 하부 지지재 305~1829mm
동바리 간격 수직 지지재 305~1829mm
상승각 보 구조 경사 지지재 280~550mm

* 표시된 범위는 일반적인 참고값이며, 현장 조건에 따라 달라질 수 있습니다.

안전성 검토

  • OK 구조적으로 안전
  • NG 재검토 필요

하중 계산

자동으로 계산되는 하중:

  • 콘크리트 하중 (24 kN/m³)
  • 거푸집 하중 (0.4 kN/m²)
  • 활하중 (두께별 차등)
  • 충격하중 (1000mm 초과 시)

구조물 타입별 사용법

슬래브 (Slab)

입력 항목

  1. 타설 두께 - 100~3500mm (슬라이더 조정)
  2. 프로젝트 타입
    • 건축: 동바리 최대 1829mm
    • 토목: 동바리 최대 1524mm
  3. X방향 스팬 - 100~15000mm
  4. Y방향 스팬 - 100~15000mm

사용 방법

1. 타설 두께 슬라이더로 원하는 두께 설정 2. 건축/토목 중 프로젝트 타입 선택 3. 슬래브 크기(X, Y 스팬) 입력 4. 자동 계산된 결과 확인

일반 보 (Normal Beam)

입력 항목

  1. 보폭 - 300~2000mm
  2. 보춤 - 300~2000mm
  3. 보 스팬 - 100~10000mm
  4. 보 타설 두께 - 100~보춤 이하
  5. 멍에 타입 - 단멍에/장멍에

멍에 타입 선택

  • 단멍에 - 보 직각 방향 배치
  • 장멍에 - 보 평행 방향 배치 (상승각 포함)

데크 보 (Deck Beam)

일반 보 설정에 추가로:

  • 슬래브 두께 - 100~250mm (데크슬래브)

고급 기능

수동 간격 설정

멍에/동바리 간격 수동 선택

  1. 자동 계산된 값 아래 "사용 가능 규격" 확인
  2. 원하는 간격 클릭하여 수동 설정
  3. 수동 설정 시 장선 간격 자동 재계산
v1.00 신기능
멍에/동바리를 수동으로 넓게 설정하면, 장선 간격이 자동으로 줄어들어 구조 안전성을 확보합니다.

동바리 배치 설정 (보 구조물)

  1. "4. 동바리 간격 계산" 영역 클릭
  2. 열 수 선택: 2열, 3열, 4열
  3. 동바리 간격 선택
  4. 확인 버튼 클릭

다중 구조물 관리

구조물 추가

  1. 상단 탭에서 "+ 추가" 클릭
  2. 새 구조물 타입 선택
  3. 각 구조물별로 개별 설정 가능

구조물 전환

  • 탭 클릭으로 구조물 간 전환
  • 현재 설정은 자동 저장

프로젝트 저장/불러오기

저장

  1. 툴바의 "💾 저장" 클릭
  2. 프로젝트 이름 입력
  3. 로컬 저장소에 저장

불러오기

  1. "📁 불러오기" 클릭
  2. 저장된 프로젝트 목록에서 선택
  3. 모든 구조물 정보 복원

보고서 및 출력

참고 보고서 생성

  1. "📄 참고 보고서 추출" 버튼 클릭
  2. 자동 생성된 보고서 확인
  3. 인쇄 또는 PDF 저장

보고서 포함 내용

  • 프로젝트 개요
  • 각 구조물별 설계조건
  • 사용재료 명세
  • 설계하중 계산
  • 구조 검토 결과

3D 시각화

  1. "🎯 3D 시각화" 버튼 클릭
  2. 마우스 조작
    • 드래그: 회전
    • 스크롤: 확대/축소
  3. 구조물 구성 요소 확인

정식 구조계산서 의뢰

중요
현장 사용을 위해서는 반드시 구조기술사의 정식 구조계산서가 필요합니다.
  1. "📋 구조기술사 의뢰하기" 클릭
  2. 의뢰인 정보 입력
  3. 제출 방법 선택
    • 이메일 제출
    • 오픈카카오톡 제출

문제 해결

자주 묻는 질문

Q: 계산 결과가 NG로 표시됩니다.

입력값을 조정하거나, 수동으로 간격을 줄여보세요.

Q: 3D 뷰가 표시되지 않습니다.

최신 브라우저 사용 및 하드웨어 가속 활성화를 확인하세요.

Q: 저장한 프로젝트를 찾을 수 없습니다.

브라우저의 로컬 저장소를 확인하세요. 쿠키 삭제 시 데이터가 손실될 수 있습니다.
