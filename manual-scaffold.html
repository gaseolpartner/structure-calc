1. 프로그램 소개

개요

시스템 비계 구조계산 프로그램은 건설 현장에서 사용되는 시스템 비계와 강관 비계의 구조 안전성을 KDS 기준에 따라 정밀하게 계산하는 웹 기반 프로그램입니다.

중요 고지사항
본 프로그램의 계산 결과는 참고용 보고서이며, 현장 조건에 따라 상이할 수 있습니다.
실제 시공 시에는 반드시 HJ구조사무실의 정식 구조계산서를 받으시기 바랍니다.
본 프로그램의 보고서는 관공서 제출용으로 사용할 수 없습니다.

주요 특징

  • 실시간 구조계산 - 입력값 변경 시 즉시 안전성 검토
  • KDS 기준 적용 - 최신 한국설계기준 적용
  • 다중 구조물 지원 - 최대 10개 구조물 동시 관리
  • 지역별 풍속 데이터 - 전국 200여개 지역 기본풍속 내장
  • 참고 보고서 생성 - 구조계산 결과 즉시 출력

구분 시스템 비계 강관 비계
수직재 Φ48.6×2.3t (SGT355) Φ48.6×2.3t (SGT355)
수평재 Φ42.7×2.2t (SGT275) Φ48.6×2.3t (SGT275)
허용하중 42.5 kN 30.0 kN
기본간격 1817mm 1817mm

2. 시작하기

프로그램 실행

  1. 웹 브라우저에서 시스템비계 구조계산 (가설파트너&HJ구조).html 파일 실행
  2. Chrome, Edge, Firefox 등 최신 브라우저 사용 권장
  3. 인터넷 연결 없이도 사용 가능

초기 화면

비계 시스템 선택
프로그램 시작 시 다음 중 하나를 선택합니다:
  • 🏗️ 시스템 비계 - OK System 등 규격화된 비계
  • 🔧 강관 비계 - 일반 강관 조립식 비계

인터페이스 구성

화면 구성
• 왼쪽 모니터: 입력 파라미터 및 계산 결과
• 오른쪽 모니터: 부재 규격 및 설치 상세
• 상단 툴바: 저장, 불러오기, 인쇄 등 기능
• 구조물 탭: 다중 구조물 관리

3. 주요 기능

자동 안전성 검토

프로그램은 다음 5가지 항목을 실시간으로 검토합니다:

검토 항목 설명 판정 기준
수직재 압축응력 수직 부재의 압축응력 검토 355 MPa 이하
수평재 휨하중 수평 부재의 휨 저항력 검토 11.9 kN 이하 (시스템)
작업발판 휨강도 발판의 처짐 및 강도 검토 허용하중 이하
벽이음 인장력 벽이음철물의 인장력 검토 37.2 kN 이하
수직재 1본의 하중 안전율 3.0 적용 허용하중/3.0 이하

상태 표시

OK (안전)
모든 구조 검토 기준을 만족하는 상태입니다.
NG (재검토 필요)
일부 검토 기준을 초과했습니다. 입력값을 조정하세요.

하중 계산

고정하중

  • 작업발판 자중: 0.2 kN/m²
  • 비계 부재 자중: 자동 계산

활하중 (작업하중)

  • 경작업: 1.0 kN/m²
  • 중작업: 2.0 kN/m²
  • 돌붙임 공사: 3.5 kN/m²

풍하중

  • 기본풍속: 지역별 20~44 m/s
  • 노풍도: A, B, C, D
  • 충실률: 0.3 (안전망), 0.5 (수직보호망)

4. 비계 시스템별 사용법

시스템 비계

특징

  • 안전인증을 받은 규격화된 제품
  • 수직재 허용하중: 42.5 kN
  • 연결부 높이: 1725mm
  • 기본 수평간격: 1817mm

사용 방법

  1. 초기 화면에서 "시스템 비계" 선택
  2. 설치 간격 입력
    • 띠장 방향 간격: 300~3000mm
    • 장선 방향 간격: 300~1200mm
  3. 작업 조건 설정
    • 작업 하중 선택
    • 작업 가능 층수 선택
  4. 풍하중 조건 입력
  5. 벽이음철물 간격 설정

강관 비계

특징

  • 일반 강관을 사용한 조립식 비계
  • 수직재 허용하중: 30.0 kN
  • 모든 부재 Φ48.6×2.3t 사용
  • 더 큰 허용 휨하중: 15.0 kN
주의사항
강관 비계는 시스템 비계보다 허용하중이 낮으므로, 동일 조건에서 더 촘촘한 간격이 필요할 수 있습니다.

5. 파라미터 설정 가이드

설치 간격

띠장 방향 간격 (수평간격)

  • 일반적으로 1817mm 사용
  • 하중이 클 경우 간격 축소
  • 최소 300mm ~ 최대 3000mm

장선 방향 간격

  • 일반적으로 902mm 사용
  • 발판 규격에 맞춰 조정
  • 최소 300mm ~ 최대 1200mm

작업 조건

작업 하중 선택 기준

구분 하중 적용 작업
경작업 1.0 kN/m² 도장, 청소, 검사 등
중작업 2.0 kN/m² 일반 건축공사, 철근 조립
돌붙임 3.5 kN/m² 석재 공사, 중량물 취급

작업 가능 층수

  • 높이 15m 이하: 3개층
  • 높이 25m 이하: 2개층
  • 높이 25m 초과: 1개층

풍하중 조건

지역 선택

  1. 드롭다운에서 해당 지역 선택
  2. 자동으로 기본풍속 설정
  3. 직접 입력도 가능

노풍도 선택

구분 설명
A 대도시 중심부
B 도시 지역, 교외 지역
C 개활지, 해안 지역 (일반적)
D 해안, 섬 지역

벽이음철물 간격

설치 기준
• 수평간격: 띠장 간격의 1~3배
• 수직간격: 3.8m (2개층) 권장
• 최대 간격: 5.7m × 7.6m

6. 고급 기능

다중 구조물 관리

구조물 추가

  1. 상단 탭에서 "+ 추가" 클릭
  2. 비계 타입 선택 (시스템/강관)
  3. 각 구조물별 개별 설정
  4. 최대 10개까지 추가 가능

구조물 전환

  • 탭 클릭으로 즉시 전환
  • 각 구조물의 설정값 자동 저장
  • 구조물별 계산 결과 개별 관리

프로젝트 저장/불러오기

프로젝트 저장

  1. 툴바의 "💾 저장" 버튼 클릭
  2. 프로젝트 이름 입력
  3. 모든 구조물 정보 포함하여 저장
  4. 브라우저 로컬 저장소에 보관

프로젝트 불러오기

  1. "📁 불러오기" 버튼 클릭
  2. 저장된 프로젝트 목록에서 선택
  3. 모든 구조물 및 설정값 복원
주의
브라우저 쿠키 삭제 시 저장된 프로젝트가 삭제될 수 있습니다.

구조물 요약 보기

"📊 구조물 목록" 버튼을 클릭하면 프로젝트의 모든 구조물을 한눈에 볼 수 있습니다.

  • 각 구조물의 주요 설정값
  • 안전성 검토 결과
  • 빠른 편집 기능

7. 보고서 및 출력

참고 보고서 생성

보고서 생성 방법

  1. 모든 입력값 확인 및 검토
  2. "📄 참고 보고서 추출" 버튼 클릭
  3. 보고서 내용 확인
  4. 인쇄 또는 PDF 저장

보고서 포함 내용

  • 프로젝트 개요 및 일반사항
  • 적용 규격 및 표준
  • 사용재료 명세
  • 설치 간격 및 조건
  • 하중 조건 및 계산
  • 구조 검토 결과
  • 가새재 설치 계획
  • 주의사항
중요
생성된 보고서는 참고용이며 관공서 제출용으로 사용할 수 없습니다.
정식 구조계산서가 필요한 경우 구조기술사 의뢰가 필요합니다.

정식 구조계산서 의뢰

의뢰 절차

  1. "📋 구조기술사 의뢰하기" 버튼 클릭
  2. 의뢰인 정보 입력
    • 회사명/의뢰인
    • 담당자 및 연락처
    • 현장명 및 위치
  3. 제출 방법 선택
    • 이메일 제출
    • 오픈카카오톡 제출
  4. 의뢰서 자동 생성 및 다운로드

정식 구조계산서의 필요성

  • 관공서 제출용
  • 구조기술사 날인
  • 법적 책임 보장
  • 현장 특수 조건 반영

8. 문제 해결

자주 묻는 질문

Q: 계산 결과가 NG로 표시됩니다.

A: 다음을 확인하세요:

  • 설치 간격을 줄여보세요
  • 작업 하중을 낮춰보세요
  • 벽이음철물 간격을 줄여보세요
  • 작업 가능 층수를 줄여보세요

Q: 저장한 프로젝트를 찾을 수 없습니다.

A: 브라우저의 로컬 저장소를 확인하세요. 쿠키 삭제나 시크릿 모드 사용 시 데이터가 저장되지 않습니다.

Q: 특정 지역이 목록에 없습니다.

A: "직접 입력"을 선택하고 인근 지역의 풍속을 참고하여 입력하세요.

Q: 보고서를 관공서에 제출할 수 있나요?

A: 아니요. 본 프로그램의 보고서는 참고용이며, 관공서 제출용 정식 구조계산서는 구조기술사를 통해 발급받으셔야 합니다.

오류 해결

프로그램이 작동하지 않을 때

  1. 최신 버전의 웹 브라우저 사용 확인
  2. JavaScript 활성화 확인
  3. 브라우저 캐시 삭제 후 재실행
  4. 다른 브라우저에서 실행 시도
